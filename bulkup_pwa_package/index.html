벌크업 가이드

5개월 10kg | 55kg → 65kg | 하루 3,000kcal
핵심 원칙
  1. 시계가 아니라 "기상 후 시간"으로 식사 타이밍 잡기
  2. 하루 4끼 + 간식, 칼로리 분산해서 먹기
  3. 고밀도 음식(PB, 견과류, 올리브오일)으로 적게 먹어도 칼로리 UP
  4. 매주 같은 재료를 사서, 같은 메뉴를 반복 (루틴이 핵심)
  5. 배달 음식은 고정 메뉴 2~3개만 돌려서 고민 최소화
  6. 단백질 90~110g: 계란 + 치킨 + 그릭요거트 + 우유 조합
  7. 깔끔한 벌크업 → 월 1~1.5kg, 하루 2,600~2,800kcal로 조절
